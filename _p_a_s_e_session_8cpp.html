Detailed Description

This file implements the CHIP SPAKE2P Session object that provides APIs for constructing spake2p messages and establishing encryption keys.

The protocol for handling pA, pB, cB and cA is defined in SPAKE2 Plus specifications. (https://www.ietf.org/id/draft-bar-cfrg-spake2plus-01.html)
